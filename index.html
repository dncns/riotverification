Boundr – User Flow

This page outlines the initial user flow for Boundr, a companion app for Riot’s Riftbound TCG.

Splash Screen

  • App logo + loading animation
  • Initializes data sync from Riftbound API

Home / Dashboard

  • Quick access to Card Database, Decks, and Collection
  • Featured deck of the week / meta highlight
  • Search bar at top

Card Database

  • Scrollable grid of cards (art + cost + name)
  • Filters: Type, Set, Cost, Rarity, Keyword, Energy Type
  • Card detail modal with:
    • Full art
    • Stats
    • Rulings
    • Flavor text
    • Related cards (combos, synergies)

Deck Builder

  • Add cards from database via + button
  • Deck list view (sorted by cost)
  • Card count tracker (total cards / per cost)
  • Save, export, or share deck

Collection Tracker

  • Toggle between Owned / Missing cards
  • Progress bar showing completion % per set
  • Auto-update from user input (manual) or account sync (future feature)

Offline Mode

  • Cached database
  • Saved decks & collection data accessible without internet
